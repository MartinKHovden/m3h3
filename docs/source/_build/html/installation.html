Installation¶

In this section different methods for obtaining m3h3 is presented.

Installation using Docker containers¶

The M3H3 package can be installed using Docker. This will set up a container with dolfin, cbcbeat, fenics, pulse, geometry and m3h3. Run all commands in the terminal.

Should change docker file for development.

  1. Download docker from docker.com for your platform.

  2. Clone

  3. Build the docker image using the Dockerfile by running

    docker build -t m3h3 docker/

  4. The next steps are similar to before. Run the docker container from the m3h3 folder to share this with the container.

    docker run -ti -v $(pwd):/home/fenics/shared --name m3h3_container m3h3

  5. Next install m3h3 in the container

    python3  setup.py install

  6. Now you have m3h3 installed as well as dolfin, fenics, cbcbeat, pulse and geometry. Enjoy!

  7. Now you can exit the container using

    exit

    This will send you back to your original system.

  8. To restart the container, use

    docker start m3h3
docker exec -ti -u fenics m3h3 /bin/bash -l

    which will send you back into the docker shell.

Installation using Anaconda¶

The M3H3 package can also be installed using Anaconda. This requires a linux system. If you are working on Windows, set up windows subsystem for linux, and run the same steps as below.

  1. First, make sure that you have conda installed on your system.

  2. Next, set up a conda environment

    conda create --name m3h3 python=3.7.3

  3. Activate the environment using

    conda activate m3h3

  4. Run the bash script to install the packages

    bash install.sh

  5. Now you have set up an environment with fenics, dolfin, cbcbeat, pulse geometry, and m3h3. Enjoy!

Installation for developers¶

Both Docker and Anaconda can be used for developing.

Docker¶

  1. Download docker from docker.com for your platform.

  2. Build the docker image using the Dockerfile by running

    docker build -t m3h3 docker/DockerfileDevelop

  3. Next run the docker container from the m3h3 folder to share this with the container.

    docker run -ti -v $(pwd):/home/fenics/shared --name m3h3_container m3h3

  4. Next install m3h3 in the container

    python3 setup.py develop

  5. Now you have m3h3 installed as well as dolfin, fenics, cbcbeat, pulse and geometry. Enjoy!

  6. Now you can exit the container using

    exit

    This will send you back to your original system.

  7. To restart the container, use

    docker start m3h3
docker exec -ti -u fenics m3h3 /bin/bash -l

    which will send you back into the docker shell.

Anaconda¶

  1. First, set up an environment

    conda create --name m3h3-develop python=3.7.3

  2. Activate the environment using

    conda activate m3h3-develop

  3. Run the bash script to install the dependencies

    bash install_dev.sh

  4. Fork the m3h3 depository to your own repo and clone it

    git clone  https://github.com/YOUR_GITHUB/m3h3.git
cd m3h3
python setup.py develop
